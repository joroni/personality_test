A bit about this demo
This demo aims to show how to use angularjs ui-router in combination with a ui requirement wherein some pages need a sidebar navigation and others don't. You'll note that this page has no side navbar and uses nested content, whereas the Products page includes a collapsible sidebar navigation with nested content areas for each sub-nav items.
